## React Native Matsuri Webサイト構築
* 2021年4月〜2021年9月、2022年9月
* React Native JP運営メンバーとして

### 自分が関わっている主な技術スタック・フレームワークなど
TypeScript/NextJS/Firebase

### プロジェクトについて
オンラインカンファレンスを行うためのWebサイト構築。  
LP -> LP第2段 -> 参加応募フォーム -> タイムスケジュールなどの本公開という感じだったと思うので、稼働時間としては短め。  
NextJSのカンファレンスみたいにGitHub連携をしてシェア用の画像を生成しようという案があったのでcanvasを用いての実装をしたが、2021年時の配信はhoppinで行うという話になり、hoppin上で応募もしないといけないという二度手間があったので参加応募フォームをなくす方向性にしたために、シェア用画像生成についてはお蔵入りとなった。  
2022年開催時にはタイムテーブルを入れ替える形でデザインは変えずに公開をした。
